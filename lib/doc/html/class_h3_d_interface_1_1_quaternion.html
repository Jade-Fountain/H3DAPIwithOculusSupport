Detailed Description

The Quaternion class represents quaternion objects that are used for rotation operations.

Operators supported:

q1 + q2 Quaternion addition.
q1 - q2 Quaternion subtraction.
q1 * v Rotation of vector(assumes q1 is normalized). v can be Vec3f or Vec3d
q1 * x Multiplication with scalar value.
q1 / x Division with scalar value.
-q1Negation.
