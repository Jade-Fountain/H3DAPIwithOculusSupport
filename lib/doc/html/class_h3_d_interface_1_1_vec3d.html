Detailed Description

The Vec3d class represents a 3d vector type using double precision floating point values for its components.

Operators supported:

v1 + v2Component-wise addition.
v1 - v2Component-wise subtraction.
x * v1, v1 * x Component-wise multiplication where x is a scalar value.
v1 * v2 Multiplication between to Vec3d types is defined as the dot product between the two vectors.
v1 % v2 Cross product.
v1 / x Component-wise division where x is a scalar value.
-v1Component-wise negation.
