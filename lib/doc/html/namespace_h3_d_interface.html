H3DInterface.createX3DFromString (   s)

Create a local H3D scene graph from the scene description specified in a string.

The string can be in X3D/XML or X3D/VRML format.

Parameters
sstring with the scene description.
Returns
A tuple with a Group containing the top nodes of the file and a dictionary with all the DEF nodes in the file for later access. On error an exception will be thrown.

Example usage:

1 s = """<Shape>
2  <Appearance> <Material DEF="MAT"/> </Appearance>
3  <Sphere radius="0.01" />
4 </Shape>"""
5 
6 group, dn = createX3DFromString( s )
7 # Set color of sphere to red.
8 dn["MAT"].diffuseColor.setValue( 1, 0, 0 )

def H3DInterface.findNodes (   node,
  type_names,
  node_name = "",
  field_names = [],
  exactNodeName = True,
  verbose = False 
)

Find nodes matching the specified criteria.

Recursively searches for child nodes that match the given criteria, starting from the specified root node.

Parameters
nodeThe root node to start the search from
type_namesType names of nodes to find, as a list of strings
node_nameThe name (or partial name) of the nodes to find (also see exactNodeName parameter)
field_namesList of field names to recursively look for children, for each node type. For each node type, the list will contain a sub-list of strings, where the first string is the type name of the node, and the remaining strings are field names to search. If field_names is [] then all fields will be included in the search.
exactNodeNameIf True then the node_name must match exactly the node's name, otherwise the node name must contain node_name as a sub-string.
verboseIf true then the function prints debugging information about the search to the console.
Returns
A list containing 1 tuple for each node found. Each tuple contains the found node, followed by a tuple containing all the nodes parents. I.e., [(found_node,(parent0,parent1,...)),...]
