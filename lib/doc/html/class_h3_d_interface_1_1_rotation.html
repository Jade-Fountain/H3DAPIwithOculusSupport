Detailed Description

The Rotation class describes an arbitrary rotation.

It specifies an axis to rotate around and the angle to rotate.

Operators supported:

r1 * r2 Concatenation of rotations.
r1 * v Rotation of vector. v can be Vec3f or Vec3d
r1 * x Angle is multiplied by x. x is a scalar value.
r1 / x Angle is divided by x. x is a scalar value.
-v1Angle is negated.
