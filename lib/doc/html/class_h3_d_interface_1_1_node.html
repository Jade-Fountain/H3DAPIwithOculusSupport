Detailed Description

Python encapsulation of an H3D Node instance.

Any node type is encapsulated by this class. Public fields from the encapsulated node can be accessed through attributes of the Python Node instance.

Example:

1 n = createX3DNodeFromString( "<Sphere />" )[0]
2 # access the radius field of the sphere node by node attribute.
3 n.radius.setValue( 0.2 )
