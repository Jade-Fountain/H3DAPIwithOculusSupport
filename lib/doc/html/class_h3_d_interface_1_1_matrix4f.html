Detailed Description

The Matrix4f class represents a 3x3 matrix type using single precision floating point values for its components.

Operators supported:

m1 + m2Component-wise addition.
m1 - m2Component-wise subtraction.
m1 * m2 Matrix multiplication.
m1 * v Matrix multiplication with column vector. v can be Vec3f, Vec3d, Vec4f or Vec4d. 3d vectors are assumed to have 1 in the fourth compnent.
m1 * x Component-wise multiplication where x is a scalar value.
m1 / x Component-wise division where x is a scalar value.
