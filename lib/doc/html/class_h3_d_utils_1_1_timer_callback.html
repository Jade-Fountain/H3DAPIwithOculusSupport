Detailed Description

The TimerCallback field is a field in which you can set callback functions to be called at a later time that you specify.

Example usage:

 def test( v ):
   print v
 tc = TimerCallback()
 tc.addCallback( time.getValue()+3, test, ("Hello world!",) )

This will call the function test 3 seconds from it is run.
